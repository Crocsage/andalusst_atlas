Administration Functions

Verify Logbooks


Approve/Reject logbook submissions and hand out the rewards for them. This is what you'll want to use upon some sort of group event ending.

Approvals


Approve/Reject anything that isn't a logbook! Like customizations.

Reward Teams


Give a whole lot of teams some kind of reward. This should probably be used sparingly, instead preferring the Verify Logbooks option.

Star Coin Management


Give/Take Star Coins from a Team.

Manage Merits/Strikes


Adjust a team's merits and strikes.

Add Event


Add an event and its rewards so that users can submit logbook entries for it.

Tweets and Timers
