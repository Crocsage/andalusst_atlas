Reroll Dungeon - {{dungeon_name}} F{{floor}}

HOLD IT!

Rerolling a dungeon floor is _only_ for instances where the map that was generated was bugged in some way. Only reroll a dungeon floor if something in the following list is true:

  • There are rooms which are not connected to the rest of the map
  • There are missing entities (such as no traps, resources, or enemies)
  • There is no entrance or exit. (Or stairs for a multi-floor dungeon)
  • There are doors or doorways which lead to nowhere
  • There are multiple traps/resources/enemies stacked on top of each other

Do not reroll a dungeon because you are not satisfied with its contents. Dungeons are a way to work with given constraints, and generating new ones repeatedly defeats their purpose.

Do not reroll a dungeon if you see a checkerboard tile on your map (). It's just a wall that wasn't defined with an image properly. Link your map to an admin and we'll get it fixed.

When you reroll a dungeon floor, the original floor is saved. Dr. Dos will be using it to help debug any issues with the dungeon generator.
Unecessary rerolls waste staff time!
